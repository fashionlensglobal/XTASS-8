# Project Rules and Instructions

## Sitemap Maintenance Requirement
*   **Always Update the Sitemap:** Whenever any new page, route, or screen is created in this project, you MUST automatically add it as a working link in `/src/components/Sitemap.tsx`.
*   **Categorization:** Place the new link under the appropriate section (Main Pages, Vehicles & Services, or Authentication) with an exact matching URL path.
*   **Accuracy:** The Sitemap must always stay 100% up to date and include all existing pages in the project. Do not complete a feature or create a page without performing this sitemap update.
